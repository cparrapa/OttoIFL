# OttoIFL
Mix from OttoDIY opensource Project.  
http://ottodiy.com 

## About this Library
- Arms
- HT16K33 i2c Led Matrix (mouth) ... 
- WiFi ... replace Bluetooth 
- mp3 ... trigger audio samples
- APA106 RGB LED (nose)
- MEMS microphone
- ADCTouch
- mBlocks Examples (Calibration, Smooth Criminal ...)

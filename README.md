# OttoIFL
Mix from OttoDIY opensource Project.  
http://ottodiy.com 
You can do it yours 
## About this Library
- HT16K33 i2c Led Matrix (mouth)
- ESP8266 Wifi, Spiffs, and Esp8266Audio ... replaces Bluetooth
- APA106 RGB LED (nose)
- MEMS microphone
- ADCTouch
